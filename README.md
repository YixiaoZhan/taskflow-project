# Proyecto FCT - CFGS DAW

Repositorio para la etapa de Formación en Centros de Trabajo (FCT) del ciclo formativo CFGS Desarrollo de Aplicaciones Web.

## Descripción general

Este proyecto agrupa los ejercicios y el proyecto final desarrollados durante las semanas de FCT. Contiene ejemplos de HTML, CSS, JavaScript y uso de Tailwind para maquetación, así como documentación de IA.

## Estructura por semanas

- Semana_1
  - FCT_1: Interfaz web sencilla desarrollada con HTML y CSS.
  - FCT_2: Gestor de tareas con CRUD local y función de búsqueda.
  - FCT_3: Versión del ejercicio 2 maquetada con Tailwind CSS.
  - ProyectoFinal: Versión finalizada de la página con funcionalidades extra.
- Semana_2
  - FCT_4: Documentación y recursos relacionados con IA y herramientas de apoyo.
  - ProyectoFinal_VersiónIA: Proyecto final con funcionalidades potenciadas por IA.

## Convenciones

- Cada ejercicio está autocontenido en su carpeta.
- Los recursos (imágenes, scripts, estilos) se encuentran en subcarpetas `recursos` cuando aplican.
- El proyecto final incluye configuración de Tailwind.
