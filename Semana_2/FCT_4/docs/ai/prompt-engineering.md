# Prompt Engineering: pruebas y 10 prompts útiles

En este documento se registran observaciones sobre prompt engineering y una lista de 10 prompts útiles con su razón de utilidad.

## Pruebas de prompt engineering

1. Prompt con rol específico
   - Cambia poco el contenido final en términos directos, pero ayuda a ajustar tono y formato.
   - Utilidad: aclara responsabilidades y expectativas para la IA.

2. Prompt con ejemplo
   - Mejora ligera en calidad y personalización del resultado.
   - Utilidad: un ejemplo guía a la IA hacia la estructura deseada.

3. Prompt con paso a paso
   - Mejora notable porque la IA puede razonar explicitando un flujo lógico.
   - Utilidad: facilita la comprensión del usuario y reduce omisiones.

4. Prompt con restricciones
   - Depende de la restricción; puede ser ventajoso para foco y concisión, pero también restringe la creatividad.
   - Utilidad: controla el tamaño, el estilo o el detallado del output.

## 10 prompts útiles con razón

1. "Incluye por ejemplo..."
   - Da contexto y hace que la respuesta se adapte mejor a tu situación.

2. "Explica de manera clara y sin relleno"
   - Reduce verborrea y entrega una respuesta más directa.

3. "Quiero hacer ... pero ..."
   - Proporciona contexto de necesidades o restricciones para una solución más relevante.

4. "¿Cómo harías ...? Yo estoy pensando en..."
   - Invita a la IA a comparar enfoques y justificar recomendaciones.

5. "Haz ... pero quiero que sea ..."
   - Combina acción con criterio de formato y estilo específico.

6. "Está bien, pero ¿se podría ...?"
   - Funciona como refinamiento iterativo para mejorar o ajustar soluciones.

7. "Explícalo más profundo / menos profundo"
   - Permite ajustar el nivel de detalle según tu conocimiento previo.

8. "Explícalo como si ..."
   - Cambia el registro para aclarar usando una audiencia (experto, novato, niño, etc.).

9. "Hazlo en ... pasos / menos de ..."
   - Impones límite de extensión y estructura paso a paso.

10. "¿Qué es mejor ...?"
   - Solicita comparación explícita para elegir la opción óptima.
